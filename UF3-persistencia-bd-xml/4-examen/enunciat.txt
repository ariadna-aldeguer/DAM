A partir dels codis treballats a classe, realitzar el següent programa:
Es tracta de crear una classe en codi Java que:
- Estigui comentada i sigui modular internament (+1 punt).
- Llegeixi del document bcn.xml les dades de les temperatures máximes i mínimes d'un dia en concret (+1 punt).
- Mostre per pantalla les temperatures màximes i mínimes del dia en concret (+2 punts).
- Guardi en un document anomenat Nom_temp.xml les dades seguint aquesta estructura/exemple (+2 punts):


        

<?xml version="1.0" encoding="UTF-8"?>
<consulta xpath="Consulta xpath utilitzada">
    <dia data="2022-01-18">
        <temperatura>
            <maxima>13</maxima>
            <minima>4</minima>
        </temperatura>
    </dia>
</consulta>


A tenir en compte:
- El valor del dia s'ha d'introduïr al programa com a paràmetre (primer dia, segon dia...) (+1 punt).
- El fitxer s'ha de generar en el mateix directori on es trobi l'execució del programa (sense usar cap ruta absoluta) (+0'5 punts).
- No s'ha s'usar "Entrada.ReadLine()" de Moisès, trobeu una altra forma estàndard i que serveixi igualment per aconseguir el mateix propòsit (+0'5 punts).
- Si el dia que introdueixi l'usuari no és correcte (fora de rang, un caràcter estrany...), s'ha de controlar l'error, informar per pantalla a l'usuari i finalitzar el programa immediatament (+2 punts).
Exemple d'execució correcta:

alvaro@pc$ java consultesXpath 3
Màxima -> 13
Minima -> 4
Més detalls al fitxer temperatura.xml
Fi del programa

Exemple d'execució fallida:

alvaro@pc$ java consultesXpath -3
Valor de dia incorrecte, revisa l'entrada.
Fi del programa

Exemple d'execució fallida:

alvaro@pc$ java consultesXpath prova
Valor de dia incorrecte, revisa l'entrada.
Fi del programa
